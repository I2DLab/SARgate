Utilities

The Utilities tab provides a suite of standalone cheminformatics tools that operate independently of the core library analysis workflow. These tools enable real-time structure visualisation, molecular string format interconversion, and dataset management, supporting the preparation and validation of input files for downstream R-group and substructure analyses.

Mol Draw and String Converter

Converts a molecular string representation between SMILES, SMARTS, InChI, or MolBlock formats while dynamically rendering the corresponding 2D molecular structure. The molecular depiction updates interactively upon each string modification, ensuring syntactic validity and structural interpretability across formats.

Merge SDFs

Enumerates all Structure Data Files (SDF) located in a user-selected directory. Selected SDF files are concatenated into a single consolidated SDF output, serialised into the same directory using a user-defined filename without modifying SD molecular properties.

Compare SDFs

Computes the intersection of molecular entries between two or more SDF files, reporting the number of exact structure matches shared across the selected datasets. This utility quantifies library overlap for independently generated or sourced SDF collections.

SDF ↔ CSV Converter

Performs bidirectional interconversion between SDF and CSV formats. Molecular structures are preserved via SMILES encoding, while SD properties are mapped to CSV columns or reinjected into SD tags. CSV delimiters are automatically detected to ensure robust parsing across regional settings and heterogeneous file sources.

For the SDF → CSV mode, activity fields are expanded into separate columns. For CSV → SDF mode, structured activity columns are optionally collapsed into a single SD property string to preserve semantic compatibility with analytical pipelines and external tools that expect unified activity metadata.

Conversion Logic

SDF → CSV

  • Reads SD molecules from the input file.
  • Encodes full molecular structures into a column labelled SMILES.
  • Extracts SD properties into additional columns.
  • Parses Activity and indexed Activity_* SD tags into structured univariate columns:
    • Standard Type,
    • Standard Relation,
    • Standard Value,
    • Standard Units.
  • Writes the final table to CSV in the same directory.

CSV → SDF

  • Detects the correct CSV delimiter.
  • Identifies the SMILES column.
  • If columns Standard Type, Standard Relation, Standard Value, Standard Units exist, they are merged into a single Activity SD tag as: Type Relation Value Units.
  • All non-empty CSV cells are injected as SD molecular properties named as their respective column headers.
  • Writes the final SDF file in the same directory.

ChEMBL Target Entries by Year

Retrieves the annual count of bioactivity annotations registered in the ChEMBL database for a given target identifier. This supports temporal trend inspection of structure–activity data deposition.
