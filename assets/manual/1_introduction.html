SARgate User Manual

Molecular Toolkit Chemical Space and Structure-Activity Relationship Analysis

SARgate is an open-source cheminformatics platform designed for the analysis of molecular libraries, with a strong focus on scaffold detection, R-group decomposition, Structure–Activity Relationship (SAR) analysis, and chemical space exploration.

This manual provides a complete description of the graphical interface, supported input formats, analysis workflow, and all post-analysis tools available in SARgate.

How to Use This Manual

The manual is organised into independent chapters, each accessible from the navigation panel on the left.

  • Use the sidebar to freely navigate between chapters.
  • The ? button inside the SARgate GUI (or the F1 key) opens the chapter corresponding to the currently active tab.
  • All documentation is available offline.

Scope and Audience

This manual is intended for medicinal chemists, computational chemists, and researchers working with chemical libraries and biological activity data.

Basic familiarity with molecular representations (SMILES, SDF), chemical descriptors, and SAR concepts is assumed.
