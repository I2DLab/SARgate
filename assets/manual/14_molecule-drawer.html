Molecule Drawer

The Molecule Drawer is SARgate's standalone 2D structure editor. It is designed for fast manual sketching, fragment editing, stereochemical annotation, clipboard-based exchange with external software, and quick image export. The interface is deliberately compact: most functions are controlled by a small number of drawing modes, direct mouse gestures, and keyboard shortcuts.

Main Interaction Model

The editor works by combining an active element with an optional bond or ring tool. In normal drawing mode, clicking or dragging on the canvas adds atoms, bonds, chains, or rings. In control modes such as Select, Erase, Pan, or Rotate, drawing is temporarily disabled and the mouse is reassigned to editing or navigation operations.

The toolbar is organised into compact groups: structure-management buttons, control buttons, drawing helpers, common atom buttons, periodic-table access, charge buttons, bond buttons, ring buttons, and visualisation/export controls. Most buttons are self-explanatory from their icon and tooltip; the sections below focus on behaviour, shortcuts, and less obvious interactions.

Mouse Controls

  • Left click on the canvas adds or edits according to the active tool.
  • Left click on an atom with an atom button active changes that atom to the selected element.
  • Left click on an atom with a bond tool active creates a new bond using geometry-aware placement.
  • Left drag from an atom previews and creates a bond, chain, or ring attachment interactively.
  • Left drag on empty space in Select mode draws a selection rectangle.
  • Left drag in Erase mode draws an erase rectangle.
  • Left drag in Rotate mode rotates the selected fragment, or the picked fragment if only one atom/bond is active.
  • Right drag pans the view.
  • Right click opens the canvas context menu.
  • Mouse wheel / trackpad zoom zooms around the cursor position.

Clipboard and Import/Paste Logic

The drawer accepts pasted structure data directly onto the canvas. Clipboard parsing supports common cheminformatics representations such as SMILES, SMARTS, InChI, and MolBlock. Imported structures are appended to the current canvas rather than replacing it.

When a fragment is selected, copy and cut operations serialise that selection as a MolBlock, making it possible to duplicate or transfer fragments between editor sessions and external tools while preserving the local 2D geometry.

Keyboard Shortcuts

  • Ctrl/Cmd + C: copy the current selection (in MolBlock format).
  • Ctrl/Cmd + X: cut the current selection (in MolBlock format).
  • Ctrl/Cmd + V: paste supported structure content from the clipboard onto the canvas.
  • Ctrl/Cmd + A: select everything on the canvas.
  • Ctrl/Cmd + Z: undo.
  • Ctrl/Cmd + Y or Ctrl/Cmd + Shift + Z: redo.
  • Delete or Backspace: delete the current selection.
  • Escape: clear the current selection.
  • H, C, N, O, P, S, F, I, R: activate the corresponding atom button.
  • C followed quickly by L: activates Cl.
  • B followed quickly by R: activates Br.

SMILES Display and Copying

The SMILES bar continuously reflects the current interpreted molecule. The adjacent Copy button copies that string directly to the clipboard. This is useful for immediate transfer into SARgate input fields, notebooks, spreadsheets, external editors, or database query tools.
